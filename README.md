# My Neovim Configuration

## Installation
```
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```
```
# clone this repo to .config/nvim
git clone https://github.com/JiraPit/VimConfig.git ~/.config/nvim
```
